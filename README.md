SSH Quick Connect is a lightweight Windows tool built with .NET Framework that lets you connect to any SSH server instantly — just enter your host and username, and you're in.

## Features

- One-click SSH connection that opens directly in a terminal window.
- Clean and minimal interface inspired by RDPQuickConnect.
- No dependencies, no NuGet packages — pure .NET Framework.
- Automatic update checker with one-click installer.

## Installation

1. Download the latest release from the [Releases](../../releases) page.
2. Run the installer (`SSHQuickConnectSetup.exe`) and follow the instructions.

## Usage

1. Open **SSH Quick Connect**.
2. Enter the **Host / IP** of your SSH server.
3. Enter your **Username**.
4. Click **Connect** — a terminal window will open and connect automatically.

## System Requirements

- Windows 10 or higher
- .NET Framework 4.7.2 or higher
- OpenSSH client installed (comes pre-installed on Windows 10/11)
